Carmen Rosa Quispe Reynaga

Soy una apasionada por la tecnología y la educación con experiencia en programación, diseño web, danza y música. Me encanta aprender cosas nuevas y siempre estoy buscando nuevos desafíos.

Habilidades

CSS

Creación de diseños web receptivos y adaptables. Capacidad para crear animaciones y transiciones CSS. Experiencia en la optimización del rendimiento web con CSS.

HTML

Excelente manejo de la estructura y semántica del código HTML. Capacidad para crear formularios web complejos y accesibles. Experiencia en la integración de contenido multimedia (imágenes, vídeos, etc.). Conocimiento de las últimas tecnologías web como HTML5 y Web Components.

JavaScript

Experiencia en frameworks populares como Angular. Conocimiento de bibliotecas JavaScript como jQuery, Underscore.js. Experiencia en la creación de aplicaciones web de una sola página.

C++

Sólida comprensión de los conceptos de programación orientada a objetos. Experiencia en el desarrollo de aplicaciones de escritorio y consola. Dominio de la gestión de memoria, punteros y referencias. Capacidad para escribir código C++ eficiente y robusto. Conocimiento de las bibliotecas estándar de C++ y de terceros

Python

Experto en el uso de Python para scripting, análisis de datos y aprendizaje automático. Dominio de las bibliotecas estándar de Python. Capacidad para escribir código Python limpio, legible y bien organizado. Conocimiento de frameworks de desarrollo en Python
